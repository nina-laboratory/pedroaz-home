# Pedroaz.Home Documentation

## Summary
Personal home page for Pedro Azevedo, showcasing work experience, projects, and contact information.

## Architecture
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Hosting**: Vercel (target)
